AI

Effort 1

Effort is a small and lightweight on-device text-text only AI application, it performs well on both CPU and GPU, this generalist model is designed for a general understanding of language, good for any daily task purposes, general text generation, and basic programming.

Endeavor 1

Endeavor is an on-device text-text only AI application with higher precision, making this model more powerful & versatile model and larger in size, designed for a wide range of tasks, making it a great choice for users with access to more resources. This model is quite CPU-friendly, but I recommend you using this model with any GPU you have for better performance.

Both models are designed to run locally on your device, ensuring your data remains private and secure. No internet connection is required, making them ideal for offline use.
